# ✈️ Tafels Vliegveld

Een leuk spelletje om de tafels van 1 t/m 10 te oefenen, met een vliegveld-thema.

## Spelen

Open `index.html` in de browser, of host het op GitHub Pages.

## Hosten op GitHub Pages

1. Maak een nieuwe repository op GitHub (bijv. `cj-tafels`).
2. Push deze map naar de repository:
   ```bash
   git init
   git add .
   git commit -m "Eerste versie tafels spel"
   git branch -M main
   git remote add origin https://github.com/<jouw-gebruikersnaam>/cj-tafels.git
   git push -u origin main
   ```
3. Ga op GitHub naar **Settings → Pages**.
4. Kies bij *Source* de branch `main` en folder `/ (root)`.
5. Na een paar minuten staat het spel op `https://<jouw-gebruikersnaam>.github.io/cj-tafels/`.

## Hoe te spelen

- Kies een tafel (1 t/m 10) of klik op **Door elkaar** voor een mix.
- Beantwoord 10 sommen. Het vliegtuig rolt over de baan met elke goede som.
- Aan het einde krijg je sterren ⭐ en zie je welke sommen fout waren.

## Bestanden

- `index.html` — structuur van de pagina
- `style.css` — vrolijke stijl met wolkjes en startbaan
- `app.js` — spel-logica

Veel plezier met oefenen! 🛫
